# 사진 넣기

이 폴더에 아래 파일명으로 사진을 넣으면 청첩장에 자동으로 표시됩니다.

- `hero.jpg` — 첫 화면 헤더 사진
- `hero-optimized.jpg` — 모바일 표시용 자동 최적화본 (원본을 교체하면 다시 생성)
- `hero-demo.jpg` — 현재 시안에 사용하는 무료 샘플 헤더 사진
- `gallery-01.jpg` ~ `gallery-07.jpg` — 갤러리 사진

파일명이나 개수를 바꾸고 싶다면 `src/content/invitation.js`의 `gallery.images` 목록을 수정하세요. JPG, PNG, WebP도 사용할 수 있습니다.
